# 🎯 GymMatch - 새로운 UI/UX 구조

## 📱 4개 메인 카테고리

### 1. Feed (홈 - 인스타그램 스타일)
**경로**: `/` 또는 `/feed`
**설명**: 로그인 후 바로 보이는 메인 페이지
**UI**: 세로 스크롤 피드
**내용**:
- 매칭된 파트너들이 업로드한 운동 사진/영상
- 좋아요, 댓글 기능
- 무한 스크롤

**현재 상태**:
- ✅ `/feed` 페이지 존재
- ✅ 사진 업로드 기능 있음
- ✅ 좋아요/댓글 기능 있음

**필요한 수정**:
- ❌ 로그인 후 `/discover`로 가는 것을 `/feed`로 변경

---

### 2. Discover (매칭 - 스와이프 스타일)
**경로**: `/discover`
**설명**: 스와이프로 운동 파트너 찾기
**UI**: Tinder 스타일 카드 스와이프
**내용**:
- 한 번에 한 명씩 카드로 표시
- 좌우 스와이프 또는 ❌/❤️ 버튼
- 매칭 시 "It's a Match!" 모달

**현재 상태**:
- ✅ `/discover` 페이지 존재
- ✅ Like/Pass 버튼 있음
- ⚠️ 프로필 안 보이는 문제 (디버깅 중)

**필요한 수정**:
- ⚠️ 프로필 로딩 문제 해결

---

### 3. Matches (채팅)
**경로**: `/matches`
**설명**: 매칭된 사람들과 채팅
**UI**: 채팅 앱 스타일
**내용**:
- 매칭된 사람 리스트
- 1:1 채팅
- 사진 공유

**현재 상태**:
- ✅ `/matches` 페이지 존재
- ❓ 채팅 기능 확인 필요

**필요한 수정**:
- ❓ 테스트 후 결정

---

### 4. Profile (프로필 & 설정)
**경로**: `/profile`
**설명**: 내 프로필 관리
**UI**: 설정 앱 스타일
**내용**:
- 프로필 사진 변경
- Bio 수정
- 운동 정보 수정
- 로그아웃

**현재 상태**:
- ✅ `/profile` 페이지 존재
- ✅ EditProfileModal 있음

**필요한 수정**:
- ❓ 테스트 후 결정

---
