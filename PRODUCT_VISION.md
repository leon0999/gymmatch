# 🏋️ GymMatch - Product Vision & Design Principles

## 🎯 핵심 디자인 철학
**"웹사이트 시작부터 끝까지 모든 스텝이 섹시하고 매력적인 디자인이고 아이덴티티가 부드럽다는 느낌을 받을 수 있게 전문 논문 참고와 지식 학습을 반복해서 전문적이고 사업적인 결과물을 만들어줘. 순차적으로 진행해서 사용자 입장에서 에러가 안나오게 하는 건 당연한 거고."**

---

## 📱 4대 핵심 페이지

### 1. Feed (피드) - 인스타그램 스타일
**목적**: 파트너들의 운동 사진 공유 및 동기부여

**핵심 기능**:
- ✅ 파트너들이 운동 중/후에 찍은 사진
- ✅ 상대 파트너가 올리면 자동으로 피드에 표시
- ✅ 모든 사용자 사진이 **시간 순서대로 정렬**
- ✅ 깔끔하고 매력적인 레이아웃

**디자인 요구사항**:
- Instagram 스타일 세로 스크롤
- 사진이 돋보이는 큰 카드
- 좋아요/댓글 기능 직관적
- 부드러운 스크롤 애니메이션

---

### 2. Discover (매칭) - 프로필 발견 및 스와이프
**목적**: 매력적인 운동 파트너 찾기

#### 🎴 프로필 카드 레이아웃/디자인 (최우선!)
**중요도**: ⭐⭐⭐⭐⭐

**필수 정보 (헬스인이 중요하게 생각하는 것)**:
1. **오늘 운동할 부위** (예: 가슴, 등, 하체, 어깨)
2. **평소 메인 기구** (예: 벤치프레스 100kg, 스쿼트 120kg)
3. **운동 중량** (주요 운동별 1RM 또는 작업 중량)
4. **운동 시간** (예: 오후 7-9시, 아침 6-8시)
5. **운동 컨디션** (예: 초급, 중급, 고급)

**추가 핵심 기능**:
- 💡 **운동 부위 건의 기능**: "오늘 하체 같이 하실래요?" 같은 메시지 전송

**디자인 요구사항**:
- ✨ **섹시하고 매력적인** 프로필 카드
- 🎭 **부드러운 스와이프 애니메이션** (Tinder 스타일 개선판)
- 📊 운동 정보가 **한눈에** 들어오는 레이아웃
- 🎨 운동하는 사람 vs 이제 시작한 사람 **차별화 표시**
- 💪 **매칭하고 싶게** 만드는 비주얼 임팩트

**스와이프 경험**:
- 좌/우 스와이프 시 부드러운 애니메이션
- 좋아요 시 하트 효과
- Pass 시 X 효과
- 슈퍼라이크 옵션 (선택사항)

---

### 3. Matches (채팅) - 깔끔한 소통
**목적**: 매칭된 파트너와 운동 약속 잡기

**핵심 기능**:
- ✅ **원활하고 깔끔한** 채팅 UI
- ✅ 실시간 메시지 전송/수신
- ✅ 운동 일정 제안 기능 (선택사항)

**디자인 요구사항**:
- 카카오톡/iMessage 스타일 심플함
- 빠른 응답성
- 사진 공유 가능

---

### 4. Profile (프로필) - 나의 운동 기록 전시
**목적**: 상대가 올린 사진 및 내 운동 기록 보기

**핵심 기능**:
- 📸 **상대 사용자들이 올려준 사진들** 잘 정렬
- 📱 **인스타그램처럼 섹시하게** 그리드 레이아웃
- 📊 피드들이 잘 보이게 타임라인

**디자인 요구사항**:
- Instagram 프로필 그리드 스타일
- 3열 사진 그리드
- 탭: 사진 그리드 / 타임라인 피드
- 프로필 헤더: 운동 통계 (총 운동 일수, 파트너 수 등)

## 🎨 전체 디자인 시스템

### 색상 팔레트
```
Primary: Teal (#14B8A6) - 활력, 신선함
Secondary: Deep Blue (#0F172A) - 전문성, 신뢰
Accent: Orange (#F97316) - 열정, 에너지
Background: Light Gray (#F8FAFC) - 깨끗함
Text: Dark (#1E293B) - 가독성
```

### 타이포그래피
```
Headings: Bold, 임팩트 있는 큰 글씨
Body: 가독성 높은 중간 크기
Labels: 작지만 명확한 정보
```

### 애니메이션 원칙
```
- 모든 전환은 부드럽게 (300-500ms)
- 스와이프는 물리적 느낌 (spring animation)
- 로딩은 스켈레톤 UI로
- Micro-interactions로 피드백 강화
```

### 레이아웃 원칙
```
- 여백 충분히 (breathing room)
- 정렬은 일관되게
- 중요한 정보는 눈에 띄게
- 계층 구조 명확히
```

## 🚀 개발 우선순위

### Phase 1: 핵심 UX 완성 (현재)
1. ✅ 회원가입/온보딩 완료
2. ✅ Discover 기본 매칭 작동
3. ⚠️ **Discover 프로필 카드 디자인 개선** ← 다음 작업
4. ⚠️ **스와이프 애니메이션 추가**
5. ⚠️ **운동 정보 필드 추가** (오늘 부위, 중량, 시간 등)

### Phase 2: 소셜 기능 강화
6. Feed 페이지 완성 (사진 업로드, 정렬)
7. 채팅 기능 완성
8. Profile 그리드 레이아웃

### Phase 3: 고급 기능
9. 운동 부위 건의 기능
10. 운동 통계 대시보드
11. 알림 시스템

## 📊 성공 지표

### 사용자 경험
- 에러율: 0% (완벽한 작동)
- 프로필 카드 → 매칭 전환율: 30%+ 목표
- 채팅 응답률: 80%+

### 디자인 품질
- "섹시하다" 느낌 받는 사용자: 90%+
- "부드럽다" 느낌 받는 사용자: 90%+
- 앱 재방문율: 70%+

### 비즈니스
- DAU (일일 활성 사용자): 1,000명 (3개월 내)
- 매칭 성공률: 60%+
- 운동 약속 성사율: 40%+

## 💡 핵심 인사이트

### 헬스인이 중요하게 생각하는 것
1. **진지함**: 운동에 대한 진지한 태도
2. **레벨**: 비슷한 수준의 파트너 선호
3. **목표**: 같은 목표 (벌크업, 다이어트, 유지 등)
4. **스케줄**: 운동 시간대 일치 여부
5. **부위**: 같은 부위 운동 시 시너지

### 차별화 요소
- **운동 정보 중심**: 다른 소셜 앱과 달리 운동에 집중
- **실용적 매칭**: 실제로 만나서 운동할 파트너
- **동기부여**: 서로의 운동 사진으로 자극
- **섹시함 + 전문성**: 감성적이면서도 체계적인 UX
